# Third-party datasets and assets

This repository's code and original documentation are separate from any third-party datasets used during experimentation or model training.

## Important
Third-party datasets are **not relicensed** by this repository.
Users are responsible for obtaining those datasets separately and complying with their original terms.

## Mentioned datasets

### Chars74K
Public dataset page:
https://teodecampos.github.io/chars74k/

The public Chars74K page states that by downloading and using the dataset, users agree to acknowledge the source and cite the associated paper in related publications.

### Kaggle Sudoku Image Dataset
Dataset page:
https://www.kaggle.com/datasets/mexwell/sudoku-image-dataset

This repository references that dataset as part of project history/benchmark context.
Users should review the dataset page directly for the current Kaggle license and usage terms before downloading or using it.
